InfoFilms

Application React permettant de consulter des films via l’API The Movie Database (TMDb) et de gérer une wishlist.

Fonctionnalités :
- Liste de films par catégories
- Recherche de films
- Page de détail d’un film
- Ajout / suppression de films dans une wishlist

Wishlist persistée avec le localStorage

1) Lancer le projet

2) Créer un fichier .env à la racine du projet :
-> VITE_TMDB_API_KEY=VOTRE_CLE_API

3) Lancer les commandes :
npm install
npm run dev
